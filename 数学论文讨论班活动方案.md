# Landmark Papers Seminar

## 一、活动简介

本讨论班主要面向求真书院本科生，讨论班计划每周举办一次，每次围绕一篇具有重要影响的数学论文展开，由一位主讲人进行报告，时长在1-1.5h。本讨论班希望通过持续阅读重要论文，帮助大家更深入地理解数学问题的来源、理论发展的脉络以及重要方法的形成过程。

## 二、活动目标

1. 阅读和理解数学各方向中的重要论文；
2. 了解经典问题、核心概念和重要方法的发展脉络；
3. 提高数学论文阅读能力和学术表达能力；
4. 通过讨论促进不同方向同学之间的交流；
5. 形成稳定的数学学习和讨论氛围。

## 三、活动对象

本讨论班主要面向求真书院本科生开放，并不拘泥于某个方向，欢迎不同研究方向的同学参加，参与者不需要完全熟悉每篇论文所属领域。

## 四、活动形式

讨论班计划除考试周外每周一次，每次选读一篇重要数学论文。每次由一位主讲人负责报告，提前一周给一份参考材料，供感兴趣的同学提前阅读学习。每次活动时长为1-1.5h。

## 五、论文选择范围

讨论班选读的论文主要包括以下类型：

1. 在某一数学领域中具有奠基性或代表性意义的论文；
2. 引入重要概念、理论框架、方法或技术的论文；
3. 解决著名问题或推动某一方向发展的论文；
4. 能体现某一数学方向核心思想的论文；
5. 对后续研究产生持续影响的经典或现代论文。

论文方向参考ICM报告方向，包括但不限于：逻辑、代数、数论、代数几何与复几何、几何、拓扑、李理论、分析、动力系统、偏微分方程、数学物理、概率、组合数学、计算机科学中的数学、数值分析与科学计算、控制理论与优化、统计与机器学习、随机与微分建模、数学史等。

## 六、主讲人安排

每次活动由一位同学担任主讲人，会提前一周以pdf形式提供报告内容，包含以下信息：

1. 论文题目、作者、年份和pdf；
2. 论文简介；
3. 所需预备知识；
4. 建议提前阅读的部分；
5. 相关参考资料。

可供参考的论文简介内容：

1. 论文研究的问题是什么；
2. 这个问题为什么重要；
3. 论文的主要结果是什么；
4. 论文使用了什么关键思想或方法；
5. 哪些证明或构造最值得关注；
6. 论文对后续研究产生了什么影响。

主讲人不一定讲完论文中的所有细节，重点是讲清楚论文的主线、核心思想和关键内容。

报告结束后，主讲人提供一份slides或手稿供参阅使用，手稿获取方式见讨论班网站。

## 七、参与方式

讨论班信息会提前一周在求真科协公众号发布，同时更新在讨论班网站。本讨论班无需报名即可参与，如有同学愿意成为主讲人，欢迎与我们联系。
